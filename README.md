# Отсчёт до Нового Года

Простой красивый статический сайт с обратным отсчётом до следующего Нового Года и конфетти при наступлении.

Файлы:
- `index.html` — основная страница
- `css/styles.css` — стили
- `js/script.js` — логика отсчёта и конфетти

Как открыть локально:

1. Просто откройте файл `index.html` в браузере (двойной клик).

2. Или запустите локальный HTTP-сервер (рекомендуется для корректной работы в некоторых браузерах):

```
# Powershell
python -m http.server 8000

# затем откройте в браузере:
http://localhost:8000/
```

Настройка и доработка:
- Меняйте стиль в `css/styles.css`.
- Меняйте поведение конфетти в `js/script.js`.
